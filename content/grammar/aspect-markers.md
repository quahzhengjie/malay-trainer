title: Aspect markers
level: B1

Because Malay verbs never change for tense, small **aspect markers** placed
before the verb do that work instead.

| Marker | Meaning | Example |
|---|---|---|
| sudah / telah | already, have done | Saya sudah makan. |
| sedang | -ing, in progress | Saya sedang makan. |
| akan | will, going to | Saya akan makan. |
| belum | not yet | Saya belum makan. |
| pernah | ever, have done before | Saya pernah ke Melaka. |
| masih | still | Saya masih lapar. |
| baru | just (recently) | Saya baru sampai. |

`telah` is the more formal cousin of `sudah` — common in writing and news.
