title: Verbs with me-
level: A2

The **`me-`** prefix builds *transitive* verbs — verbs that take an object.
Its shape changes with the first sound of the root, and that first sound is
sometimes dropped. This is called **nasalisation**.

| First letter of root | Prefix | Example |
|---|---|---|
| l, m, n, r, w, y | me- | lihat → melihat |
| b | mem- | baca → membaca |
| p | mem- (p drops) | pukul → memukul |
| d | men- | dengar → mendengar |
| t | men- (t drops) | tulis → menulis |
| s | meny- (s drops) | sapu → menyapu |
| g, h, vowel | meng- | ambil → mengambil |
| k | meng- (k drops) | kira → mengira |

It looks like a lot of rules, but you absorb the pattern by ear with practice.
