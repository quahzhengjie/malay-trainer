Conjunctions link two parts of a sentence.

| Malay | English |
|---|---|
| dan | and |
| atau | or |
| tetapi / tapi | but |
| kerana / sebab | because |
| jadi / oleh itu | so, therefore |
| walaupun / meskipun | although |
| supaya / agar | so that |
| kalau / jika | if |
| semasa / ketika | when, while |

*tapi* and *sebab* are the everyday spoken forms; *tetapi* and *kerana* are the
standard written forms.
